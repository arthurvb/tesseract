1. Install Visual Studio 2015 & CMake
2. Install [vcpkg](https://github.com/Microsoft/vcpkg/) 
	* Note: I also set an environment variable VCPKG_HOME to this directory and added it to path for convenience
	
3. Build Leptonica:
	```
	vcpkg install giflib:x86-windows-static libjpeg-turbo:x86-windows-static liblzma:x86-windows-static libpng:x86-windows-static tiff:x86-windows-static zlib:x86-windows-static
	git clone https://github.com/DanBloomberg/leptonica.git & cd leptonica	
	mkdir vs15-x86 & cd vs15-x86
	cmake .. -G "Visual Studio 15 2017" -DCMAKE_TOOLCHAIN_FILE=%VCPKG_HOME%\scripts\buildsystems\vcpkg.cmake -DVCPKG_TARGET_TRIPLET=x86-windows-static -DCMAKE_INSTALL_PREFIX=..\..\build
	cmake --build . --config Release --target install
	```
	
	Note: In order to build the leptonica dll statically you'll also need to patch CMakeLists.txt to include liblzma:
	
	CMakeLists.txt:
	
	Add ``find_package(LIBLZMA)`` at line 55 (after find_package(ZLIB).
	
	src\CMakeLists.txt:
	
	Add:
	```
	if (LIBLZMA_LIBRARY)
		target_include_directories  (leptonica PUBLIC ${LIBLZMA_INCLUDE_DIR})
		target_link_libraries       (leptonica ${LIBLZMA_LIBRARY})
	endif()
	```
4. Build Tesseract:
	```
	vcpkg install leptonica
	git clone https://github.com/tesseract-ocr/tesseract.git
	cd tesserct
	mkdir vs15-x86 & cd vs15-x86
	cmake .. -G "Visual Studio 15 2017" -DCMAKE_INSTALL_PREFIX=..\..\build
	cmake --build . --config Release --target install
	```
